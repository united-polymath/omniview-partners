# OmniView Partners

Marketing website for **OmniView Partners** — a premium, data-driven coaching firm.

Static site built with HTML + Tailwind (CDN) + vanilla JS. No build step required.

## Pages
- `index.html` — Home
- `about.html` — About & OmniView Labs
- `method.html` — The OmniView Method
- `platform.html` — Platform (includes Programs)
- `experts.html` — Experts / Coaches
- `partners.html` — Partners
- `contact.html` — Start free assessment
- `privacy.html` / `terms.html` — Legal

## Local preview
Just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deployment
Deployed via GitHub Pages from the repository root (`main` branch).
